 ### [319. Bulb Switcher](https://leetcode.com/problems/bulb-switcher/description/)

### Only for small n Time O(n^2)
```
class Solution {
    public int bulbSwitch(int n) {
        int arr[] =new int[n];
        for(int i=1;i<=n;i++){
            for(int j=i-1;j<n;j+=i){
                arr[j]^=1;
            }
        }
        return countbulb(arr);

        
    }
    int countbulb(int[] arr){
        int bulb=0;
        for(int v:arr){
            if(v==1){bulb++;}
        }
        return bulb;
    }
}

```
