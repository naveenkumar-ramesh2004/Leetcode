## [1423. Maximum Points You Can Obtain from Cards](https://leetcode.com/problems/maximum-points-you-can-obtain-from-cards/description/)

```
class Solution {
    public int maxScore(int[] arr, int k) {
        int total=0;
        int min=0;
        int minSum=0;
        for(int i=0;i<arr.length-k;i++){
            min+=arr[i];
        }
        minSum=min;
        total+=min;
        for(int i=arr.length-k;i<arr.length;i++){
            min+=arr[i];
            min-=arr[i-(arr.length-k)];
            minSum=Math.min(min,minSum);
            total+=arr[i];
        }
        return (k==arr.length)?total:total-minSum;
    }
}
