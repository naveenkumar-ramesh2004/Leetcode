## [53. Maximum Subarray](https://leetcode.com/problems/maximum-subarray/description/)

# 53. Maximum Subarray

## Pattern

**Kadane's Algorithm**

## How to Recognize

* Find the **maximum sum** of a contiguous subarray.
* Subarray can start and end anywhere.
* Need an optimal solution in `O(n)` time.
* Negative numbers are present.

## Intuition

At every index, ask:

```text
Should I extend the current subarray?
OR
Should I start a new subarray from here?
```

If the current sum becomes negative, it will only decrease future sums.

Therefore, discard it and start a new subarray.

## Example

```text
[-2,1,-3,4,-1,2,1,-5,4]
```

```text
Current Sum:

-2 → reset
1  → max = 1
1+(-3) = -2
4  → start new subarray
4+(-1) = 3
3+2 = 5
5+1 = 6  ← maximum
```

Maximum Subarray:

```text
[4,-1,2,1]
```

Maximum Sum:

```text
6
```

## Algorithm

1. Maintain `currentSum` and `maxSum`.
2. Add the current element to `currentSum`.
3. Update `maxSum`.
4. If `currentSum` becomes negative, reset it to `0`.
5. Continue until the end of the array.

## Complexity

| Operation | Complexity |
| --------- | ---------- |
| Time      | O(n)       |
| Space     | O(1)       |

class Solution {
    public int maxSubArray(int[] nums) {
        int max=Integer.MIN_VALUE;
        int sum=0;
        for(int val:nums){
            sum=Math.max(val,sum+val);
            max=Math.max(max,sum);
        }
        return max;
    }
}
