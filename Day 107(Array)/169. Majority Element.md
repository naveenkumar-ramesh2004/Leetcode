# 169. Majority Element

## Pattern

**Moore's Voting Algorithm**

## How to Recognize

* An element appears more than `n / 2` times.
* Majority element is guaranteed to exist.
* Need `O(n)` time and `O(1)` extra space.

## Intuition

Think of different elements as cancelling each other out.

Example:

```text
[2,2,1,1,1,2,2]

2 cancels 1
2 cancels 1
1 cancels 2

Remaining candidate = 2
```

Since the majority element occurs more than half of the array size, it cannot be completely cancelled by the remaining elements. Therefore, the element left at the end must be the majority element.
```
class Solution {
    public int majorityElement(int[] nums) {
        int ans=nums[0];
        int count=0;
        for(int val:nums){
            if(ans==val){
                count++;
            }
            else{
                count--;
                if(count==0){
                    ans=val;
                    count=1;
                }
            }
        }
        return ans;
    }
}
