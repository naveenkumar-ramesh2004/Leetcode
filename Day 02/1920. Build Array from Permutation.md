## [1920. Build Array from Permutation](https://leetcode.com/problems/build-array-from-permutation/)
```

class Solution {
    public int[] buildArray(int[] nums) {
        int lengtharr=nums.length;
        int[] ans =new int[lengtharr];
        for(int i=0;i<lengtharr;i++){
            ans[i]=nums[nums[i]];
        }
        return ans;
        
    }
}
