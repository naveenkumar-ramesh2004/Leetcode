## [1929. Concatenation of Array](https://leetcode.com/problems/concatenation-of-array/description/)

```
class Solution {
    public int[] getConcatenation(int[] nums) {
        int n=nums.length;
        int[] ans=new int[2*(n)];
        System.arraycopy(nums,0,ans,0,n);   
        System.arraycopy(nums,0,ans,n,n);   
        return ans;
    }
}
