### [796. Rotate String](https://leetcode.com/problems/rotate-string/)


```
class Solution {
    public boolean rotateString(String s, String goal) {
        for(int i=0;i<s.length();i++){
            String value=s.substring(i)+s.substring(0,i);
            if(goal.equals(value)){
                return true;
            }
        }
        return false;
    }
}
