## 55.[Jump Game](https://leetcode.com/problems/jump-game/description/)

```
class Solution {
    public boolean canJump(int[] nums) {
        int j=nums[0];
        for(int i=1;i<nums.length;i++){
            if(j+i>=nums.length){
                return true;
            }
            if(j>0){
                j--;
                if(j<nums[i]){
                    j=nums[i];
                }
            }
            else{
                return false;
            }
        }
        return (j>=nums.length-1);
    }
}
