##[1470. Shuffle the Array](link: https://leetcode.com/problems/shuffle-the-array/description/)


```
class Solution {
    public int[] shuffle(int[] nums, int n) {
        int[] ans=new int[nums.length];
        
        for(int i=0;i<nums.length;i+=2){
            ans[i]=nums[i/2];
            ans[i+1]=nums[n++];
        }
        return ans;
       
        
    }
}
