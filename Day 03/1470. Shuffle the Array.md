## [1470. Shuffle the Array](https://leetcode.com/problems/shuffle-the-array/description/)

###Approch 1
```
class Solution {
    public int[] shuffle(int[] nums, int n) {
        int[] ans=new int[nums.length];
        
        for(int i=0;i<nums.length;i+=2){
            ans[i]=nums[i/2];
            ans[i+1]=nums[n++];
        }
        return ans;
       
        
    }
}
```
##Approch 2

```
class Solution {
    public int[] shuffle(int[] nums, int n) {
        int[] ans = new int[2*n];
        int idx=0;
        for(int i=0;i<n;i++){
            ans[idx++]=nums[i];
            ans[idx++]=nums[i+n];
        }
        return ans;
    }
}
